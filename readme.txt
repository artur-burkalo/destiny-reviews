The plugin to show reviews/testimonials on your website which will help generate more leads and makre more conversions.

== Description ==
 
The plugin will help to build better brand awareness and will let to share success stories with your users (aka humans).

Use [destiny_reviews] where you want to show reviews on your website.

Use [destiny_reviews_user_form] where you want users to submit their form.

If you do spot any bugs please email artur@destiny.ie and I'll try to fix it.

You can choose between three styles: dark, light and custom dark is default, you can also change the border radius.

The plugin is lightweight, and only loads CSS where the shortcode is preseneted.

Tested with 
- Oxygen Builder
- Custom Themes
- Gutenberg Editor
 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
1. Upload `destiny-reviews.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add where you want to show your reviews with a shortcode  [destiny_reviews]
4. Add where you want to show review submissions form with a shortcode  [destiny_reviews_user_form]
